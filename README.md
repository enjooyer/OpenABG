## ğŸš€ Installation

### Prerequisites

- **Node.js 22+** (required)
- **pnpm** (or use `npx pnpm`)
- **Git**

### Step 1: Clone the Repository

```bash
git clone https://github.com/enjooyer/OpenABG.git
cd OpenABG
```

### Step 2: Install Dependencies

```bash
npx pnpm install
```

### Step 3: Build the Project

```bash
npx pnpm build
```

### Step 4: Initialize Configuration

```bash
npx pnpm start setup
```

This creates `~/.openabg/openabg.json` and initializes your workspace.

    "apiKeys": {
      "openai": "sk-...",
